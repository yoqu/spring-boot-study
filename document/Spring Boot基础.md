##  5.1 Spring Boot概述

### 5.1.2 Spring Boot核心功能
 1. 独立运行spring项目

        可以以jar包的形式独立运行
 2. 内嵌Servlet容器

        可以选择内嵌tomcat,Jetty或者Undertow
 3. 提供start简化Maven配置

 4. 自动配置Spring

 5. 准生产的应用监控

        可以基于http ssh telnet对运行时的项目进行监控
 6. 无代码生成和xml配置

    Spring Boot通过条件注解来实现,spring boot中不需要任何xml配置即可实现Spring的所有配置
